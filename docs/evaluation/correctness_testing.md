### All
- Return a complete tour

### Brute Force
- Generate all tour permutations
- Returns tour of shortest length

### Branch and Bound
- Returns tour of shortest length

### Approx Min Span Tree

- Returns a tour with length at most 1.5x optimal

### DFJ
